# NOMINA ADVISOR BACK

1. Clone project

   `https://github.com/CarlosDanielGR/nomina-advisors-back.git`

2. Install dependencies

   `yarn install | npm install`

3. Raise the database (Check the **.env** file to copy the password)

   `docker-compose up`
